Resumo de aula - 19 de Abril

Nessa aula, foi ensinado como se utiliza seletores de css, ou seja, como selecionar uma tag específica, ou grupos de tags para personalizar. Como exemplos temos o seletor universal (*), seletor de classe (.) e id (#), além de seletores filhos e descendetes, sendo exibido também quando se deve utilizá-los. Ainda na linha de propriedades do css, foi exibido conceitos de borda, margem e preenchimento em relação ao conteúdo e também a questão de padronização de tamanhos, através de pixels, em, porcentagem e rem.

A tag <form>, que é de extrema importância para páginas web, também foi apresentada. Ela é utilizada para o preenchimento de dados informados por usuários que passam por um processamento do servidor após o envio. Dentro de um formulário, várias entradas podem recebids pela tag <input>, que muda seu tipo de dado recebido pelo atributo "type". Nessa aula foram apresentados tipos como o text, password, checkbox, radio, além de botões como button, submit e reset, e ainda três outras tags: <select>, utilizada para gerar lista uma de seleção, <label>, para nomear os campos de entrada, e <textarea> para textos maiores.
