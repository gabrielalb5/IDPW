Resumo de aula - 28 de Junho

A aula ministrada em 28 de junho contou com atividades de revisão de conteúdo de tudo que havia sido ministrado até o momento. Um dos tópicos abordados foram os conceitos básicos da linguagem de marcação usada pra desenvolvimento de páginas web, o HTML, e formas de validação, que ajudam a classificar e posicionar a página com uma qualidade maior de acordo com o uso de metadados ou !DOCTYPE, além de questões de estrutura e semântica. No HTML também existem tags, que são responsáveis por incluir o conteúdo de uma página. Existem tags de listas ordenadas (<ol>) ou não (<ul>), imagens (<img>) e até mesmo tags que ajudam na criação de formulários (<form>). Tags possuem atributos, que são palavras usadas dentro das tags de abertura para controlar o comportamento desse elemento. Em formulários, os atributos são extremamente importantes pois podem definir que tipo de conteúdo essa tag vai receber, como texto, número, áudio, vídeo.

O CSS, ou a folha de estilo em cascata, também esteve presente na revisão. Ela é responsável por armazenar estilos da página HTML. Existem três modos de vincular o estilo à página: inline, dentro da própria tag HTML; interno, através da tag <style> no cabeçalho da página; e externo, por meio de um arquivo separado referenciado no cabeçalho, o que é mais eficiente para manutenção e modificações. O CSS utiliza seletores para aplicar estilos aos elementos HTML, e o modelo de caixa é um conceito essencial para entender como o espaço ao redor dos elementos é calculado, abrangendo margens, bordas, preenchimentos e o conteúdo. A sintaxe básica do CSS é composta por três partes principais: seletor, indicando qual elemento será personalizada (pode ser tag, classe, id); propriedade, o aspecto do estilo que você deseja aplicar (cor, tamanho, fonte); e valor, que é basicamente o valor referente à propriedade definida (cor -> azul, tamanho -> 15px, fonte -> sans-serif).
