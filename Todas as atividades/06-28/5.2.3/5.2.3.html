Principais diferenças entre HTML 4 e HTML 5

Aspecto
Estrutura e Semântica Usava <div> e <span> para layout e estrutura. Introduziu elementos semânticos como <header>, <footer>, <article>, <section>, <nav>, e <aside>.
Formulários Conjunto básico de elementos de formulário. Novos tipos de entrada como email, url, range, date, color, entre outros. Melhorias em validação e usabilidade.
Multimídia Requeria plugins externos (ex: Flash) para áudio e vídeo. Introduziu <audio> e <video> para suporte nativo a mídia.
Canvas e SVG Não tinha suporte nativo para gráficos dinâmicos. Inclui o elemento <canvas> para gráficos dinâmicos e aprimorou o suporte a SVG.
APIs e Integração Suporte limitado para JavaScript e APIs externas. Novas APIs como geolocalização, armazenamento local (localStorage e sessionStorage), histórico, e arrastar e soltar.
Desempenho e Armazenamento Utilizava cookies para armazenamento e tinha mecanismos de cache limitados. Introduziu Web Storage API para armazenamento local e sessões, e Cache API para recursos offline.
Acessibilidade Suporte limitado para acessibilidade. Melhorou a acessibilidade com novos elementos semânticos que auxiliam tecnologias assistivas.
Compatibilidade com Dispositivos Móveis Não otimizado para dispositivos móveis. Melhor compatibilidade com dispositivos móveis, suporte a modo offline e construção de aplicações web responsivas.
Doctype Requeria uma declaração de doctype mais complexa. Doctype simplificado (<!DOCTYPE html>) para facilidade de uso e memorização.
