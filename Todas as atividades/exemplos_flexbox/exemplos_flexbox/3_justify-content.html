Exemplos de Utilização da Propriedade Justify-content

Justify-content: flex-start

A
B
C

Os blocos são alinhados ao início do contêiner.

Justify-content: flex-end

A
B
C

Os blocos são alinhados ao final do contêiner.

Justify-content: center

A
B
C

Os blocos são centralizados no contêiner.

Justify-content: space-between

A
B
C

Os blocos são distribuídos com espaço igual entre eles, com o primeiro bloco no início e o último bloco no final do contêiner.

Justify-content: space-around

A
B
C

Os blocos são distribuídos com espaço igual ao redor de cada bloco.

Justify-content: space-evenly

A
B
C

Os blocos são distribuídos com espaço igual entre todos os blocos, incluindo as extremidades do contêiner.
